# VPN Landing

Современный одностраничный лендинг под VPN/личный сервер с темно-серым фоном и акцентом #0048FF. Реализована анимация, адаптивы для телефонов/планшетов/ноутбуков/десктопов и классическая таблица сравнения.

## Требования
- Node.js 18+

## Скрипты
- `npm install` — установка зависимостей
- `npm run dev` — запуск в режиме разработки (Vite)
- `npm run build` — сборка production
- `npm run preview` — предпросмотр собранной версии

## Структура
- `src/App.jsx` — разметка страницы
- `src/styles.css` — стили и анимации (Tailwind CDN подключен в `index.html` для утилит)
- `dist/` — результат сборки (`npm run build`)

## Цветовая тема
- Фон: темно-серый `#0f1117`
- Акцент: синий `#0048FF` (без циана)
